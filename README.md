## About

Generated by [Leihua Ye](https://github.com/LeihuaYe) using Claude Code with web search. Each digest covers:

- **AI Progress** — model releases, architecture breakthroughs, lab announcements, open-source developments
- **AI Evaluation** — new benchmarks, eval methodology, LLM-as-judge research, leaderboard updates, safety evals
- **This Week's Highlight** — the single most impactful development for practitioners building or evaluating AI systems
